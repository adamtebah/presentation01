[적용 사례 - 첫 웹사이트]

  • 내 블로그에 링크시킬 index.html 파일을 제미나이와의 대화를 통해서 만들다.
  • "만들어진 index.html 을 네이버 블로그에서 사용하려면 어떻게 하는 것이 가장 최적일까" 물었더니, 직접 붙여 넣는 것은 불가능하고, 대안으로 아래와 같이 제시하였다.
    1. 이미지 캡처하여 배너로 사용
    2. 핵심 내용과 링크를 블로그 게시물로 옮겨 편집
    3. 가장 중요한 버튼을 노션 페이지 링크나 카카오톡 채널 링크로 대체
    4. 무료 호스팅 서비스 GitHub Pages 이용
  • GitHub 에 나만의 웹사이트를 만들고 네이버 블로그에 홈페이지 개설에 대한 링크를 만들었다.
